# Contributing to best-practices

Thanks for improving our recommendations. Please follow these guidelines:

- Keep entries concise and actionable.
- Provide a short rationale and an example when appropriate.
- Add tests or checks where appropriate (lint rules, CI snippets).
- When adding a new guideline, include `examples/` demonstrating the practice.

To contribute:

1. Fork the repository.
2. Create a feature branch.
3. Add your guideline and example.
4. Open a pull request describing the change.
